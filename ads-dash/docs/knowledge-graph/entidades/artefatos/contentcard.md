# ContentCard

## Tipo

**Artefato**  (artifact)

## Descricao

- ContentCard is a card similar to an Ad Library that showcases previews, metrics, and engagement tags.
- ContentCard is a card style for the Ad Library that includes elements like metrics, tags, a call to action, and a viral outline in a 9:16 preview format.
- A React component responsible for rendering a content card that displays media along with various properties like author, published date, metrics, tags, and actions.
- A component that represents an individual post or content item, displaying media type, thumbnail, caption, and engagement metrics.
- A UI component for displaying content in a card format.
