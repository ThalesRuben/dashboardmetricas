# KpiCard

## Tipo

**Artefato**  (artifact)

## Descricao

- KpiCard is a versatile user interface element designed to present key performance indicators (KPIs) across various application contexts. It serves as a display component that showcases important metrics, such as the number of partners, attributed sales, and performance-related data. The KpiCard is implemented in different pages of an application, notably within the Ambassadors Page and the SeoPage, where it conveys relevant insights on both ambassador/influencer performance and SEO metrics.

The KpiCard is characterized by its visual clarity, being specifically redesigned to display KPI values in a structured format that is both user-friendly and appealing. Its purpose is to enhance the user's understanding of critical metrics and progress through a visually attractive presentation. As a React component, KpiCard is capable of handling various types of KPIs, including those related to WhatsApp metrics, such as conversations and conversion rates, thereby broadening its applicability within the digital dashboard.

Overall, KpiCard plays a crucial role in visualizing performance insights across different domains, enabling users to make informed decisions based on real-time data.
- A component in JavaScript React used to display Key Performance Indicators (KPIs) with various properties such as label, value, delta, tone, and accentColor.
- A component that represents a KPI (Key Performance Indicator) card in a user interface, styled using CSS.
- A component displaying key performance indicators such as followers, reach, engagement, and profile visits, including delta change and series data.
- A UI component for displaying Key Performance Indicators in a card format.
- A UI component that represents a key performance indicator card to display different metrics related to TikTok engagement.
- A UI component used to display key performance indicators (KPIs) in a structured way, showcasing metrics such as subscriber count and average views.
