# RoasChart

## Tipo

**Componente**  (component)

## Descricao

- A React component that renders a bar chart using the library 'react-chartjs-2' based on the provided data for today and yesterday, applying specific themes and options for presentation.
- A UI component that visualizes Return on Ad Spend (ROAS) data in chart format.
