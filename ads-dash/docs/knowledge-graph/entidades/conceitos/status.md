# Status

## Tipo

**Conceito**  (concept)

## Descricao

- The current state or condition of a campaign, such as whether it is active or paused.
- A CSS class for styling a status label with font properties and padding.
- A label component with properties for font size and weight, indicating the current state of an engine with specific styles for different statuses.
- The current operational status of the campaigns, indicating whether they are active or require review, presented in the CampaignTable.
- The current state of a campaign, indicating whether it is active or inactive.
