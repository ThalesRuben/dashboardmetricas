# BudgetChart

## Tipo

**Conteudo**  (content)

## Descricao

- A UI component that visualizes budget allocation and spending in chart format.
- A CSS module file for styling a budget chart component, containing various style rules for layout and presentation elements such as donut charts and legends.
- A React component that utilizes the Doughnut chart from the Chart.js library to visualize budget data between Meta Ads and Google Ads.
