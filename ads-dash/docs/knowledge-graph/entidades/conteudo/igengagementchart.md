# IgEngagementChart

## Tipo

**Conteudo**  (content)

## Descricao

- A React component designed to display engagement metrics using a line chart. It receives a series of data points as input props and renders a chart with labels and data representing engagement percentages over time.
- A chart component that visualizes weekly engagement metrics over a specified series.
- A UI component for rendering a chart that shows engagement metrics for Instagram posts.
