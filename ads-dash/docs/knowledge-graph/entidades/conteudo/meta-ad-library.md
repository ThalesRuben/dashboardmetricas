# Meta Ad Library

## Tipo

**Conteudo**  (content)

## Descricao

- A resource used for advertising data and information provided by Meta platforms.
- A public repository showing active advertisements from any page, including details on creatives, formats, and start dates, used for competitive analysis of advertising strategies.
- Meta Ad Library is a resource provided by Facebook where users can view active and inactive ads across Facebook's platforms.
