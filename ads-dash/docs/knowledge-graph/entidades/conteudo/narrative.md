# Narrative

## Tipo

**Conteudo**  (content)

## Descricao

- A CSS class that styles narrative sections with padding, background color, border properties, and margin.
- A narrative function that summarizes changes in financial and sales metrics between two periods, including investment, revenue, and sales performance.
- A textual summary generated from the comparison of two periods’ metrics that highlights percentage and absolute variations.
