## Descricao

- SocialLineChart is a shared component used for visualizing social media metrics.
- A React component for displaying a line or bar chart, showing the growth or volume of social networks based on a series of data inputs for dates and values.

- A React component that visualizes data using line charts, specifically for social metrics such as conversations and conversion rates.
- A UI component that visualizes data using line charts, specifically designed to represent growth trends such as follower growth or views.
