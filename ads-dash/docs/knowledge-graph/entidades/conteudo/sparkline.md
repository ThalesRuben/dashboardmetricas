# Sparkline

## Tipo

**Conteudo**  (content)

## Descricao

- Sparkline refers to an embedded SVG representation that shows trends for KPIs over the last seven days without utilizing significant space.
- A component used within KpiCard to visually represent a series of data points in a line graph format.
- A React component that generates a mini inline sparkline graph using pure SVG, designed for visual trends without taking up much space, commonly used within KPI cards.
- A visual representation component used within the Metric Explainer to display trends over time in a compact format.
