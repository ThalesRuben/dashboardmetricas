# accentColor

## Tipo

**Dado**  (data)

## Descricao

- A property of the KpiCard component that specifies a custom color for accents, overriding the tone property.
- AccentColor is a prop in the Tabs component that defines the color used for styling the selected tab's border and other elements, providing a visual cue for users regarding the current selection.
