# active

## Tipo

**Dado**  (data)

## Descricao

- A prop in the FilterChips component indicating the currently active filter item, which is used to highlight the selected chip.
- A numeric state that indicates the currently active item index in the command list within the CommandPalette allowing keyboard navigation.
