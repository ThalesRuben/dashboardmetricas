# CONTENT_TAGS

## Tipo

**Dado**  (data)

## Descricao

- A collection or list of tags used to categorize and label the content types for better organization and analysis.
- CONTENT_TAGS is an array of content categories related to performance validation including tags like 'Antes e depois', 'Tutorial / passo a passo', 'Promoção / oferta', 'Bastidor', 'Depoimento de cliente', 'Tendência', and 'Dica / educativo'.
- A predefined set of tags that categorize the themes or topics of scripts selectable within the ScriptGenerator component.
