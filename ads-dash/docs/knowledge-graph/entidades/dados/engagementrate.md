# EngagementRate

## Tipo

**Dado**  (data)

## Descricao

- A metric represented as a percentage that indicates the level of user interaction with the Instagram post.
- A quantitative measurement of engagement rates for TikTok videos, indicating the level of interaction relative to views or followers.
