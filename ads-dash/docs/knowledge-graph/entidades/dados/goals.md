# goals

## Tipo

**Dado**  (data)

## Descricao

- An array of goal objects used to evaluate marketing performance metrics.
- A collection of objectives or targets that are managed within the MetricsContext.
- A structured set of objectives that users can define and track in the alerts system, each having attributes like enabled status, value, and unit.
