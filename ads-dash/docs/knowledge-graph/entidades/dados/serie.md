# serie

## Tipo

**Dado**  (data)

## Descricao

- An array of objects containing date and value properties, representing engagement data used by the IgEngagementChart component to visualize trends.
- A property of the KpiCard component that holds a series of data points for visual representation in a Sparkline.
- A structured representation of data points that includes dates and corresponding reach values for the Instagram content.
