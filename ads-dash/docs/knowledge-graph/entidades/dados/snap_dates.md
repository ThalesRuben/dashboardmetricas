# SNAP_DATES

## Tipo

**Dado**  (data)

## Descricao

SNAP_DATES are a collection of dates (01/04, 08/04, 15/04, 22/04, 29/04) that represent the snapshots taken approximately every five days over the last month, used to track the performance of Espaço Vip.
