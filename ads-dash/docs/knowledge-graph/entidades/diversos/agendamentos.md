- Agendamentos indicates appointments effectively scheduled in the calendar, representing metrics closely associated with actual revenue.>
- Agendamentos refers to the number of appointments scheduled through the funnel.
- A variable representing the number of appointments scheduled on a given day.
- A metric representing scheduled appointments on a weekly basis, with a current value of 50, but it is not enabled.
