# items

## Tipo

**Diverso**  (other)

## Descricao

- A collection of individual posts grouped by publication date.
- Items is an array of objects passed to the Tabs component, each containing properties such as id, label, and an optional badge, representing the individual tabs that users can interact with.
- A prop in the FilterChips component that contains a list of filter items available for user interaction.
