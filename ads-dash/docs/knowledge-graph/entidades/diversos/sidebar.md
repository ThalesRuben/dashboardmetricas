# Sidebar

## Tipo

**Diverso**  (other)

## Descricao

- Sidebar refers to the navigation panel organized into collapsible sections to reduce cognitive load for users.
- A vertical column in the user interface that typically contains navigation options or additional tools.
- A component that houses navigation elements like tabs and filters, facilitating user interaction with the application.
- A CSS class defining the style of a sidebar, including width, position, height, and background properties, designed to remain sticky.
