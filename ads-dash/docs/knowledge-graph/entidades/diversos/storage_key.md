# STORAGE_KEY

## Tipo

**Diverso**  (other)

## Descricao

- A constant key used to store and retrieve report schedule fallback data from the local storage in the browser.
- STORAGE_KEY is a constant that holds the key name for storing chat history in local storage, ensuring that the assistant can retrieve it later.
