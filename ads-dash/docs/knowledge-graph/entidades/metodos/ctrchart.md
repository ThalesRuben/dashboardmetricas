# CtrChart

## Tipo

**Metodo**  (method)

## Descricao

- A UI component that visualizes Click-Through Rate (CTR) metrics in chart format.
- A React functional component that renders a line chart using the 'react-chartjs-2' library. It accepts data as props and utilizes the 'useChartTheme' hook for theming.
