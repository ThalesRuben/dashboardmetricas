# Metric

## Tipo

**Metodo**  (method)

## Descricao

- An individual unit of measurement within the metrics system, often containing values that represent specific data points.
- A function component that displays a specific icon, value, and label depicting a metric in a structured format.
- A Metric represents a single unit of measure within the metrics component, structured for clarity and consistency in showing performance or engagement indicators.
- A component that styles metric data with a background, border-radius, and padding for visual appearance.>
- A React component used to display the individual metrics of a competitor, such as followers and engagement.
