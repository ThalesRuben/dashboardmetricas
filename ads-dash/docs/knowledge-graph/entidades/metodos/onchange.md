# onChange

## Tipo

**Metodo**  (method)

## Descricao

- OnChange is a function prop provided to the Tabs component that is invoked when a user selects a tab, allowing parent components to manage the active state and perform additional actions accordingly.
- A function prop provided to the FilterChips component that is called when a filter chip is clicked, allowing updates to the active filter.
