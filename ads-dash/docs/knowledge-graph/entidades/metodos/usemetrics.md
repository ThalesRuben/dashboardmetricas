# useMetrics

## Tipo

**Metodo**  (method)

## Descricao

- A custom hook in the back office project that handles metrics-related functionalities, including refreshing and saving daily metrics.
- A custom React hook that manages metrics loading based on a specified period and interacts with the MetricsContext.
- A custom React hook that retrieves metrics data based on the provided period.
- useMetrics is a custom hook that retrieves metrics data for the AI assistant, providing insights based on user inquiries.
- A custom React hook that manages and retrieves metrics data for different timeframes such as 'mes' (month) and 'hoje' (today).
