# useToast

## Tipo

**Metodo**  (method)

## Descricao

- A React context hook that provides a way to display toast notifications, generally used for user feedback within the application.
- A custom hook that provides toast notifications to users, allowing for user feedback on actions and events within the application.
- A custom hook that allows components to access the ToastContext and use its methods.
